This example lets the user set the difficulty level for multiplication facts, and then tests the user.:
 

<html>
<head>
   <title>Example 2</title>

<script>

function randomnumber10()
{
  return Math.ceil(Math.random()*5 + 5);
}

function randomnumber20()
{
  return Math.ceil(Math.random()*10 + 10);
}

function randomnumber5()
{
  return Math.ceil(Math.random()*5);
}

function randomnumber(radio)
{
  if (radio[0].checked)
  {
      return randomnumber5();
  }
  if (radio[1].checked)
  {
      return randomnumber10();
  }
  if (radio[2].checked)
  {
      return randomnumber20();
  }
}

function testSolution(a, b, c, d, e)
{
  if (a * b == c)
  {
     d.value++;
     return "Correct!"
  }
  else
  {
     e.value++;
     return "Wrong!"
  }
}

function getproblem(level)
{
 document.form1.number1.value = randomnumber(level);
 document.form1.number2.value = randomnumber(level);
 document.form1.answer.value = ""
 document.form1.feedback.value = ""
}

</script>

</head>
<body onload="getproblem(document.form1.level);">

<br><input TYPE="radio" NAME="level" VALUE="Easy" CHECKED>Easy

What's new here is the use of radio buttons. Each radio button that has the same name is part of an array.

The checked property of a radio button returns whether it has been checked (true or false).

The next example is a bit fancier.

<html>
<head>
   <title>Example 4</title>
<script>

function gender(radio, life)
{
  if (radio[0].checked)
  {
      life.value = 75;
  }
  if (radio[1].checked)
  {
      life.value = 80;
  }
}
 
function smoker(radio, life)
{
  if (radio[0].checked)
  {
      life.value = parseInt(life.value) + 5;
  }
  if (radio[1].checked)
  {
      life.value = parseInt(life.value) - 2;
  }
  if (radio[2].checked)
  {
      life.value = parseInt(life.value) - 5;
  }
}

function exercise(radio, life)
{
  if (radio[0].checked)
  {
      life.value = parseInt(life.value) - 5;
  }
  if (radio[1].checked)
  {
      life.value = parseInt(life.value) - 2;
  }
  if (radio[2].checked)
  {
      life.value = parseInt(life.value) + 2;
  }
  if (radio[3].checked)
  {
      life.value = parseInt(life.value) + 5;
  }
}

function checkbox(cb, life)
{
  if (cb.checked)
  {
      life.value = parseInt(life.value) - 10;
  }
}

function drivestyle(options, life)
{
  if (options.selectedIndex == 0)
  {
      life.value = parseInt(life.value) + 2;
  }
  if (options.selectedIndex == 1)
  {
      life.value = parseInt(life.value) - 1;
  }
  if (options.selectedIndex == 2)
  {
      life.value = parseInt(life.value) - 10;
  }
}

function calculateExpectancy(form1)
{
 gender(form1.set1, form1.life);
 smoker(form1.set2, form1.life);
 exercise(form1.set3, form1.life);
 checkbox(form1.cb1, form1.life);
 checkbox(form1.cb2, form1.life);
 checkbox(form1.cb3, form1.life);
 checkbox(form1.cb4, form1.life);
 drivestyle(form1.driving, form1.life);
}
</script>
</head>
<body bgcolor="#FFFF80">

Now we are using checkboxes and a select with options also.

Notice that the same function checkbox is being used with each checkbox. The checked property of a checkbox (like for a radio button) returns either true or false.

For a select tag, the selectedIndex property gives you the index number (starting at 0) of the option that was selected.
