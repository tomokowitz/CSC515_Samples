Place-anywhere Collapsing Menus

The menu below consists of a set of nested divs that can be placed anywhere on the page. A collapsing/expanding menu will appear when you click the + or - sign next to any menu name. This technique works in the latest versions of Internet Explorer, Netscape Navigator, and all browsers that support the W3C standard DOM and CSS.
